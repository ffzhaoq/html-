2. HTML 起手应该写:
 ```javascript
  <!DOCTYPE html>
  <html lang="zh-CN">
  <head>
      <meta charset="UTF-8">
      <meta http-equiv="X-UA-Compatible" content="IE=edge">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>网页</title>
  </head>
  <body>

  </body>
  </html>
  <!-- 该段代码可在VScode中输入html:5自动生成 -->
 ```
